AI Story Generator V6

Hindi AI Story Generator with Edge-TTS Voices.

Installation

pkg update

pkg install git python ffmpeg termux-tools -y

git clone https://github.com/nahid93/voiceapp.git

cd voiceapp

pip install -r requirements.txt

termux-setup-storage

python app.py

Features

- Hindi Story Generation
- MadhurNeural Voice
- SwaraNeural Voice
- Custom Voice Support
- Voice Preview
- Output History
- Download Last Output
- Play Last Output
